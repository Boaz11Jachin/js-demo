자바스크립트의 변수 및 자료형

자바스크립트도 데이터 유형이라는게 존재하고, 그걸 관리하기 위한 변수라는 개념도 존재한다.

자바스크립트의 기본 데이터 형은 boolean, number, string 이 존재할꺼다. 자바와 차이가 있다면, 정수,실수가 따로 없음 number 로 통일 되있음. string 은 표기할때는 홀따옴표, 쌍따옴표, 백틱 세가지 다 됨.

number 는 산술연산을 할수 있는데, 나누기 연산이 몫만 결과로 나오지 않음. (몫을 구하는게 번거로움)

string 은 + 을 지원한다. (문자열 이어만들기). + 이 아닌 연산을 하면(-, *, /, %) 를 하면 string을 number로 변환해서 계산을 해버림

변수를 쓸꺼면 let 으로 선언하면 됨 (var는 쓰지말것) ! 데이터 유형이 머냐에 따라 고민을 할 필요는 없고, 무조건 let으로 만들면 됨. 변수에 데이터가 안들어있다면 undefined 가 나올꺼임.

const 라는 걸로도 만들수 있는데, 값을 변화시키지 않을 변수를 만들때 사용. (한번 들어가면 고정)

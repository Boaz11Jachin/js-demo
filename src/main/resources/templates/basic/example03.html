자바스크립트의 제어문

제어문은 조건에 따라 분기처리를 하거나 반복처리를 할때 사용한다. 자바스크립트도 이런 제어문이 존재하고, 문법도 자바와 비슷하다. 분기처리를 하고자할때는 if ~ else (if ~ else if ~ else if else) / switch 사용하면 되고, 반복처리를 하고자할때는 for / while 을 사용하면 된다.

자바스크립트 if문

쓰는 방법자체는 비슷한데, if의 조건이 true , false 만 돌아가는건 아니다.

실제 데이터가, true 로 인식되는지 false 로 인식되는 확인하는 방법이 있다.

자바스크립트 연산자

자바스크립트도 데이터 연산이 가능하다. 자바스크립트가 제공하는 다양한 연산자에 대해 살펴보자. 대부분의 연산자들은 타 프로그래밍 언어와 유사하며, 특별히 신경써야 되는 연산자 위주로 테스트 해보며 진행해 본다.

1. 산술연산 및 증감연산

  • + : 덧셈
  • - : 뺄셈
  • * : 곱셈
  • / : 나눗셈 (몫이 나오는건 아니다)
  • % : 나머지셈
  • ** : 제곱
  • ++ , -- : 증감연산
  • +=, -=, *=, /= , %= : 배정대입연산

2.비교연산 및 논리연산

  • 크기비교 : >, >=, <=, <
  • 동등비교 : ==, ===, !=, !==
  • 논리연산 : &&, ||

3. 조건연산

  • 삼항연산 : (조건)? (값) : (값)
  • null 병합연산 : ??
